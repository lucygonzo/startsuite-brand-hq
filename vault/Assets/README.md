## Brand Guide Reference

- **Primary Font:** Plus Jakarta Sans (Google Fonts)
- **Primary Gradient:** linear-gradient(135deg, #6607E1 0%, #BC98FF 100%)
- **Dark Navy (Text):** #070322
- **Off-White (Text on Dark):** #EDEBF4
- **Design System:** Dark-mode-first, rounded corners, generous white space

For the full visual identity guide, see `/vault/03-Visual/` and the brandData.visualIdentity section in `/client/src/data/brandData.ts`.

---

## Logo Usage Guidelines

- Always maintain clear space around the logo equal to the height of the "S" icon
- Never stretch, rotate, or distort the logo
- Never apply the logo in colors other than the approved palette
- Never recreate the logo -- always use the provided files
- Minimum size: 120px wide for horizontal, 80px wide for stacked icon-only
- On photography, ensure sufficient contrast -- use the reversed version on dark images
